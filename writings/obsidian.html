Obsidian for Note-Taking

I use Obsidian for personal notes, primarily because:

  • I subscribe to Steph Ango's view on software (e.g., File over app, 100% user-supported).
  • It is free of any user-hostile dark patterns.
  • It has an extensive plugin ecosystem and high composability with any other tools I wish to use.
  • I like to write in markdown and value WYSIWYG.
Despite the pros, many end users struggle with syncing personal notes across devices without paying fees to cover server costs. Thus, I'm sharing a free, simple solution that uses git (many online tutorials are needlessly long):
  1. Create a repository on github.
  2. Generate and copy a github personal access token (classic) (set expiration=no expiration, scope=repo).
  3. Install the git community plugin in Obsidian.
  4. Create a folder to store the remote repository.
  5. Ctrl/Command + p → Git: clone an existing remote repo
  6. URL format: https://<PERSONAL_ACCESS_TOKEN>@github.com/<USERNAME>/<REPO>.git
  7. Type in the folder name to sync with the remote repo.
  8. Restart Obsidian.
  9. Make edits.
  10. Ctrl/Command + p → Git: create backup
